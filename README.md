Job Portal Platform 🚀
A modern and responsive Job Portal Platform built using React, TypeScript, Tailwind CSS, Mantine UI, and Tabler Icons.
The platform provides an intuitive interface for users to explore jobs, upload opportunities, and interact with a professional hiring ecosystem.

🚀 Getting Started
1️⃣ Clone the Repository
git clone https://github.com/Rahul12886/JOB-PORTAL.git
2️⃣ Navigate to Project Directory
cd JOB-PORTAL
3️⃣ Install Dependencies
npm install
4️⃣ Start Development Server
npm run dev

📦 Build for Production
npm run build
