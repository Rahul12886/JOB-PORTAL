# 🚀 HireNest — Smart Hiring Platform

HireNest is a modern and responsive **smart hiring platform** designed to connect recruiters with talented candidates through a clean, fast, and user-friendly interface. The platform enables users to explore job opportunities, manage hiring workflows, and experience a seamless recruitment ecosystem.

# ✨ Features

- 🔍 Smart job browsing and search functionality
- 💼 Professional hiring platform interface
- 📱 Fully responsive design for all devices
- ⚡ Fast and optimized React frontend
- 🎨 Modern UI built with Tailwind CSS & Mantine UI
- 🧩 Reusable and scalable component architecture
- 🍔 Mobile responsive navbar with hamburger menu
- 🔔 Interactive notifications and settings interface
- 🎬 Smooth animations and transitions
- 🌙 Clean and modern user experience

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Rahul12886/HireNest.git
```

## 2️⃣ Navigate to the Project Directory

```bash
cd HireNest
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Start Development Server

```bash
npm start
```

---

# 📦 Build for Production

```bash
npm run build
```
